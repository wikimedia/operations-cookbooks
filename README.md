# Cookbooks

Collection of cookbooks to automate and orchestrate operations in the WMF infrastructure.

The cookbooks will be executed by the `cookbook` entry point script of the `spicerack` package.

See the [cookbook documentation](https://wikitech.wikimedia.org/wiki/Spicerack/Cookbooks) and the [Spicerack documentation](https://doc.wikimedia.org/spicerack/master/introduction.html#cookbooks-api).
